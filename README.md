# Personal-Finance-Tracker
Mini project - Personal Finance Tracker.

In today’s fast-paced world, managing personal finances has become increasingly important. Individuals, especially students, often struggle to keep track of their income and expenses. Without proper financial tracking, it becomes difficult to control spending, save money, and make informed financial decisions.
This project titled “Personal Finance Tracker using Microsoft Excel” is designed to solve this problem by providing a structured and efficient way to record, analyze, and visualize financial data. The system enables users to monitor their financial activities in a simple yet effective manner.
Microsoft Excel is chosen as the platform for this project due to its flexibility, powerful built-in functions, and ability to handle data efficiently without requiring advanced programming knowledge.
